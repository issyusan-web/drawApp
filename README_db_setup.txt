# drawApp データベース構築手順（PostgreSQL）

このプロジェクトを動作させるためには、PostgreSQL 上に
DB「drawapp」を作成し、テーブル定義および初期データを投入する必要があります。
以下の手順に従ってセットアップしてください。

------------------------------------------------------------
■ 1. PostgreSQL の準備
------------------------------------------------------------

PostgreSQL をインストールしてください。
バージョン例：PostgreSQL 14 / 15 / 16 いずれも可。

本 README では以下の設定を前提とします：

- DB名：drawapp
- ユーザー名：postgres
- パスワード：postgres
- ポート：5432（デフォルト）


------------------------------------------------------------
■ 2. データベースの作成、テーブルの作成
------------------------------------------------------------

プロジェクト内の「sql/schema.sql」を実行して、
テーブル定義を作成します。

例：コマンドラインから実行する場合

psql -U postgres -d drawapp -f sql/schema.sql

実行後、テーブルが正しく作成されていることを確認してください。


------------------------------------------------------------
■ 3. 初期データ投入（data.sql の実行）
------------------------------------------------------------

管理者用データを投入します。

psql -U postgres -d drawapp -f sql/data.sql


------------------------------------------------------------
■ 4. アプリ側（Java）の DB 接続設定
------------------------------------------------------------

JDBC 接続情報は以下のとおりです。

- JDBC URL: jdbc:postgresql://localhost:5434/drawapp
- ユーザー: postgres
- パスワード: postgres
